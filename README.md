Программа дает скачивать web верии игр с itch.io

# Сбрка 
Установите python (https://www.python.org/downloads/).
Для сборки нужен Pyinstaller

# Нестабильность ⚠️

На данный момент прога в бете, многие игры могут не качаться. Ваш API может не поддерживать нокоротые игры

# Завасимости 📦

- `customtkinter` - Отрисовывает всё визуальное
- `requests` - Общается с интернетом, находит игру через API Itch.io и скачивает ее
- `pyperclip` - Работает с буфером обмена 
- `beautifulsoup4` - тоже для интерфейса
